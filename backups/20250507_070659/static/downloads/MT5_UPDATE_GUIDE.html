MT5 GENESIS EA Update Guide

⚠️ Why Update?

We've identified an issue where the MT5 EA is successfully connecting to our GENESIS platform but not sending trade data back. This update fixes that issue by implementing proper account status and trade data reporting.

Updating Your MT5 GENESIS EA

Step 1: Download the Updated EA

Download the updated MT5_GENESIS_EA_UPDATE.mq5 file from this page.

Step 2: Install in MetaTrader 5

  1. Open MetaTrader 5
  2. Click on File -> Open Data Folder
  3. Navigate to MQL5/Experts
  4. Copy the downloaded MT5_GENESIS_EA_UPDATE.mq5 file to this folder
  5. Rename the file to MT5_GENESIS_EA.mq5 (or keep as a separate EA if you prefer)

Step 3: Compile the EA

  1. In MetaTrader 5, press F4 to open MetaEditor
  2. Open the MT5_GENESIS_EA.mq5 file
  3. Press F7 or click the Compile button to compile the EA
  4. Ensure there are no compilation errors

Step 4: Configure the EA

  1. Ensure the API_ENDPOINT is correctly set to your GENESIS platform URL
  2. Set proper values for account name and other parameters
  3. The default value for ACCOUNT_UPDATE_INTERVAL is now 15 seconds
  4. The default value for TRADES_UPDATE_INTERVAL is now 20 seconds

Step 5: Restart the EA

  1. Drag the compiled EA onto your chart
  2. Configure input parameters as needed
  3. Click OK to start the EA

What's Changed in the Update?

  • Added regular account status updates (every 15 seconds)
  • Added regular trade data updates (every 20 seconds)
  • Improved error handling and logging
  • Fixed issues with trade data not being sent to the GENESIS platform
  • Added better signal handling for BUY_NOW and SELL_NOW actions

Troubleshooting

If you encounter any issues after updating:

  1. Check the EA's log output in the Journal tab (press Ctrl+T in MT5)
  2. Verify the API_ENDPOINT is correctly set
  3. Ensure your MT5 platform allows WebRequest for the API_ENDPOINT URL
  4. If the problem persists, please contact support
